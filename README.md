Node.js Demo
============

This is an example of using nodejs from simple HTTP server to using gruntjs + connect

#Server 1
From command line type 
`node server.js`

#Server 2
From command line install all dependencies with
`npm install`

Then run the server with
`grunt`

#Server 3
From command line install all dependencies with
`npm install`

Then run the server with
`grunt`
